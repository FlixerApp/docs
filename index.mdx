## Welkom bij de Flixer Pro API

De Flixer Pro API stelt je in staat om leads en opdrachten te beheren via een eenvoudige REST interface. Ideaal voor het automatiseren van lead capture vanaf je website of het synchroniseren van data met externe systemen.

## Aan de slag

<Steps>
  <Step title="API-sleutel aanmaken">
    Ga naar **Instellingen > API-sleutels** in je Flixer dashboard om een nieuwe sleutel aan te maken.
  </Step>
  <Step title="Authenticatie instellen">
    Voeg de `X-API-Key` header toe aan al je requests.
  </Step>
  <Step title="Eerste request maken">
    Volg onze [snelstart handleiding](/quickstart) om je eerste lead aan te maken.
  </Step>
</Steps>

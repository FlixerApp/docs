## Base URL

Alle API requests gaan naar:

```
https://api.flixerpro.io/v1
```

## Authenticatie

Alle endpoints vereisen een API-sleutel in de `X-API-Key` header:

```bash
curl -X GET "https://api.flixerpro.io/v1/leads" \
  -H "X-API-Key: jouw_api_sleutel"
```

Zie [Authenticatie](/authentication) voor meer informatie over het aanmaken en beheren van API-sleutels.

## Request formaat

- **Content-Type:** `application/json` voor POST/PUT/PATCH requests
- **Encoding:** UTF-8

## Response formaat

Alle responses zijn JSON. Succesvolle responses bevatten de gevraagde data. Foutresponses bevatten `statusCode`, `message`, en `error` velden.

## Endpoints overzicht

| Endpoint | Method | Beschrijving |
|----------|--------|--------------|
| `/leads` | GET | Lijst van alle leads |
| `/leads` | POST | Nieuwe lead aanmaken |
| `/work-items` | GET | Lijst van alle opdrachten |
| `/address` | GET | Lijst van alle adressen |

## Interactieve documentatie

Gebruik de "Try it" functie op elke endpoint pagina om requests direct vanuit de documentatie te testen. Vul je API-sleutel in en pas de parameters aan.

## Overzicht

Met de Flixer API kun je leads rechtstreeks vanaf je website naar Flixer sturen. Een lead kan bevatten:

- **Klantgegevens** - Contactinformatie van de klant (verplicht)
- **Opdrachtgegevens** - Beschrijving van de gewenste werkzaamheden (verplicht)
- **Adresgegevens** - Waar de klus uitgevoerd moet worden (optioneel)

<Note>
  Adresgegevens zijn optioneel. Zo kun je ook leads aanmaken die alleen
  naam/e-mail/telefoon bevatten, zoals bij Meta/Facebook Lead Ads en de
  CSV-import. Er wordt alleen een adres aangemaakt wanneer `street`, `postalCode`
  én `city` allemaal zijn meegegeven; een gedeeltelijk adres wordt genegeerd.
</Note>

## Endpoint

```
POST https://api.flixerpro.io/v1/api/leads
```

**Vereiste scope:** `leads:write`

## Request velden

### Adres (optioneel)

Het volledige adres is optioneel. Wil je een adres koppelen, geef dan `street`,
`postalCode` én `city` samen mee — anders wordt het adres genegeerd en krijgt de
lead `addressId: null`.

| Veld | Type | Verplicht | Voorbeeld |
|------|------|-----------|-----------|
| `street` | string | Voorwaardelijk¹ | `"Keizersgracht"` |
| `streetNumber` | string | Nee | `"123"` |
| `streetNumberAddition` | string | Nee | `"A"` |
| `postalCode` | string | Voorwaardelijk¹ | `"1015 CJ"` |
| `city` | string | Voorwaardelijk¹ | `"Amsterdam"` |
| `country` | string | Nee | `"Nederland"` |

¹ Alleen verplicht als je een adres wilt aanmaken: `street`, `postalCode` en
`city` zijn alle drie nodig of geen van alle.

### Klant (verplicht)

| Veld | Type | Verplicht | Voorbeeld |
|------|------|-----------|-----------|
| `customerName` | string | Ja | `"Jan de Vries"` |
| `firstName` | string | Nee | `"Jan"` |
| `lastName` | string | Nee | `"de Vries"` |
| `email` | string | Nee | `"jan@voorbeeld.nl"` |
| `phone` | string | Nee | `"+31612345678"` |
| `companyName` | string | Nee | `"De Vries B.V."` |

<Note>
  Als `companyName` is ingevuld, wordt de klant als zakelijke klant geregistreerd.
</Note>

### Opdracht (verplicht)

| Veld | Type | Verplicht | Voorbeeld |
|------|------|-----------|-----------|
| `title` | string | Ja | `"Lekkage badkamer"` |
| `description` | string | Nee | `"Waterdruppels op het plafond"` |
| `category` | string | Nee | `"Loodgieter"` |
| `priority` | enum | Nee | `"medium"` |
| `budgetQuote` | string | Nee | `"€500 - €1000"` |
| `notes` | string | Nee | `"Graag snel contact"` |

**Priority waarden:** `low`, `medium`, `high`, `urgent` (standaard: `medium`)

## Lead zonder adres

Heb je alleen naam/e-mail/telefoon (zoals bij Meta/Facebook Lead Ads of een
CSV-import)? Laat de adresvelden dan gewoon weg. De lead wordt aangemaakt met
`addressId: null`.

```bash cURL
curl -X POST "https://api.flixerpro.io/v1/api/leads" \
  -H "X-API-Key: jouw_api_sleutel" \
  -H "Content-Type: application/json" \
  -d '{
    "customerName": "Jan de Vries",
    "email": "jan@voorbeeld.nl",
    "phone": "+31612345678",
    "title": "Lekkage badkamer"
  }'
```

```javascript JavaScript
async function createLead(formData) {
  const response = await fetch("https://api.flixerpro.io/v1/api/leads", {
    method: "POST",
    headers: {
      "X-API-Key": process.env.FLIXER_API_KEY,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      // Adres
      street: formData.street,
      streetNumber: formData.streetNumber,
      postalCode: formData.postalCode,
      city: formData.city,
      // Klant
      customerName: `${formData.firstName} ${formData.lastName}`,
      firstName: formData.firstName,
      lastName: formData.lastName,
      email: formData.email,
      phone: formData.phone,
      // Opdracht
      title: formData.subject,
      description: formData.message,
      category: formData.category
    })
  });

  if (!response.ok) {
    throw new Error(`API error: ${response.status}`);
  }

  return response.json();
}
```

```python Python
import requests
import os

def create_lead(form_data):

        "https://api.flixerpro.io/v1/api/leads",
```

## Response

Bij succes ontvang je de aangemaakte lead:

```json
{
  "id": "WRKabc123def456ghi789",
  "brandId": "BRAabc123...",
  "customerId": "CUSabc123...",
  "addressId": "ADDabc123...",
  "title": "Lekkage badkamer",
  "description": "Klant meldt waterschade aan het plafond...",
  "status": "request",
  "priority": "high",
  "source": "web",
  "receivedAt": "2026-03-27T10:30:00.000Z",
  "createdAt": "2026-03-27T10:30:00.000Z"
}
```

<Note>
  De response bevat het volledige work item-object. Naast de bovenstaande velden
  worden ook velden als `category`, `budgetQuote`, `notes`, `estimatedCost`,
  `finalCost`, `tags`, `scheduledStart`/`scheduledEnd`, `updatedAt`, `createdBy`
  en `updatedBy` teruggegeven (leeg/`null` als ze niet van toepassing zijn).
  Is er geen (volledig) adres meegegeven, dan is `addressId` `null`.
</Note>

<Tip>
  Bewaar het `id` van de aangemaakte lead als je later de status wilt opvragen.
</Tip>

## Foutafhandeling

| HTTP Status | Betekenis |
|-------------|-----------|
| `201 Created` | Lead succesvol aangemaakt |
| `401 Unauthorized` | API-sleutel ontbreekt of is ongeldig |
| `403 Forbidden` | Sleutel heeft niet de `leads:write` scope |
| `500 Internal Server Error` | Serverfout — o.a. wanneer een verplicht veld (`customerName`, `title`) ontbreekt |

<Warning>
  Het endpoint voert momenteel **geen veldvalidatie** uit op de inhoud van de
  request. Ontbrekende verplichte velden worden niet met een nette `422`
  validatiefout afgewezen, maar leiden tot een `500`. Valideer daarom verplichte
  velden zelf vóór het versturen.
</Warning>

```javascript
async function createLeadWithErrorHandling(formData) {
  try {
    const response = await fetch("https://api.flixerpro.io/v1/api/leads", {
      method: "POST",
      headers: {
        "X-API-Key": process.env.FLIXER_API_KEY,
        "Content-Type": "application/json"
      },
      body: JSON.stringify(formData)
    });

    if (response.status === 401) {
      console.error("Ongeldige API-sleutel");
      return { success: false, error: "auth_error" };
    }

    if (response.status === 403) {
      console.error("Sleutel mist de leads:write scope");
      return { success: false, error: "scope_error" };
    }

    if (!response.ok) {
      console.error("Onverwachte fout:", response.status);
      return { success: false, error: "server_error" };
    }

    const lead = await response.json();
    return { success: true, lead };
  } catch (error) {
    console.error("Netwerkfout:", error);
    return { success: false, error: "network_error" };
  }
}
```

## Step 1: Get an API key

1. Log in to your [Flixer Pro dashboard](https://app.flixerpro.io)
2. Go to **Settings > API keys**
3. Click **Create new key**
4. Give the key a name (e.g. "Website integration")
5. Select the required scopes:
   - `leads:read` - To retrieve leads
   - `leads:write` - To create leads
6. Copy the generated key (it is shown only once!)

<Warning>
  Keep your API key secure. Never share it in public code or repositories.
</Warning>

## Step 2: Make your first request

Test your API key by retrieving the list of leads:

<CodeGroup>
```bash cURL
curl -X GET "https://api.flixerpro.io/v1/api/leads" \
  -H "X-API-Key: your_api_key"
```

```javascript JavaScript
const response = await fetch("https://api.flixerpro.io/v1/api/leads", {
  headers: {
    "X-API-Key": "your_api_key"
  }
});
const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.get(
    "https://api.flixerpro.io/v1/api/leads",
    headers={"X-API-Key": "your_api_key"}
)
print(response.json())
```
</CodeGroup>

You receive a JSON response with your leads:

```json
{
  "data": [
    {
      "id": "WRK...",
      "title": "Lekkage badkamer",
      "status": "request",
      "customer": {
        "name": "Jan de Vries"
      }
    }
  ],
  "limit": 50
}
```

## Step 3: Create a lead

Create your first lead:

<CodeGroup>
```bash cURL
curl -X POST "https://api.flixerpro.io/v1/api/leads" \
  -H "X-API-Key: your_api_key" \
  -H "Content-Type: application/json" \
  -d '{
    "street": "Keizersgracht",
    "streetNumber": "123",
    "postalCode": "1015 CJ",
    "city": "Amsterdam",
    "customerName": "Jan de Vries",
    "email": "jan@voorbeeld.nl",
    "phone": "+31612345678",
    "title": "Lekkage badkamer",
    "description": "Waterdruppels op het plafond"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.flixerpro.io/v1/api/leads", {
  method: "POST",
  headers: {
    "X-API-Key": "your_api_key",
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    street: "Keizersgracht",
    streetNumber: "123",
    postalCode: "1015 CJ",
    city: "Amsterdam",
    customerName: "Jan de Vries",
    email: "jan@voorbeeld.nl",
    phone: "+31612345678",
    title: "Lekkage badkamer",
    description: "Waterdruppels op het plafond"
  })
});
const lead = await response.json();
console.log("Lead created:", lead.id);
```

```python Python
import requests

response = requests.post(
    "https://api.flixerpro.io/v1/api/leads",
    headers={
        "X-API-Key": "your_api_key",
        "Content-Type": "application/json"
    },
    json={
        "street": "Keizersgracht",
        "streetNumber": "123",
        "postalCode": "1015 CJ",
        "city": "Amsterdam",
        "customerName": "Jan de Vries",
        "email": "jan@voorbeeld.nl",
        "phone": "+31612345678",
        "title": "Lekkage badkamer",
        "description": "Waterdruppels op het plafond"
    }
)
print("Lead created:", response.json()["id"])
```
</CodeGroup>

  Congratulations! You have created your first lead through the API.

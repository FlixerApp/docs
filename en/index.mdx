## Welcome to the Flixer Pro API

The Flixer Pro API lets you manage leads and work items through a simple REST interface. Ideal for automating lead capture from your website or synchronizing data with external systems.

## Getting started

<Steps>
  <Step title="Create an API key">
    Go to **Settings > API keys** in your Flixer dashboard to create a new key.
  </Step>
  <Step title="Set up authentication">
    Add the `X-API-Key` header to all your requests.
  </Step>
  <Step title="Make your first request">
    Follow our [quickstart guide](/en/quickstart) to create your first lead.
  </Step>
</Steps>
